# Josue Balbontin - Personal CV & JS Playground

Este proyecto es mi página personal tipo CV (Curriculum Vitae) alojada en GitHub Pages.
El objetivo principal es mostrar mi perfil profesional y, al mismo tiempo, servir como un espacio para practicar y demostrar habilidades en **JavaScript Nativo (Vanilla JS)**, HTML y CSS.

## URL del Proyecto
Puedes visitar la página en: [josue-balbontin.github.io](https://josue-balbontin.github.io)

## Características
- **Diseño Personalizado**: Estilo propio utilizando CSS.
- **JavaScript Nativo**: Implementación de funcionalidades e interactividad sin depender de frameworks externos.
- **Responsive**: Adaptable a diferentes dispositivos.

## Tecnologías
- HTML5
- CSS3
- JavaScript (ES6+)

## Estructura del Proyecto
- `index.html`: Estructura principal de la página.
- `index.css`: Estilos de la página.
- `README.md`: Documentación del proyecto.
